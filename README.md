## 📋 Описание

Проект реализует USB HID-совместимый button box для авиасимуляторов (DCS, MSFS, IL-2 и др.).  
Основа — матрица кнопок 4×4, прошивка на bare-metal C, симуляция схемы в Proteus.

**Цель:** создать полнофункциональную панель управления без использования готовых решений — только свой контроллер, своя прошивка.

## 🔧 Прошивка

- **IDE:** STM32CubeIDE
- **Язык:** C (bare-metal, без RTOS)
- **Сканирование матрицы:** построчное, polling
- **Антигостинг:** программный (фильтрация фантомных нажатий при сканировании)

### Сборка

1. Открыть папку `firmware/` как проект в STM32CubeIDE
2. Build → `Project > Build All` (Ctrl+B)
3. Прошить через ST-Link: `Run > Run` или загрузить `build/buttons.hex` через STM32CubeProgrammer

---

## 🧪 Симуляция (Proteus)

Файл `simulation/button_box.pdsprj` содержит схему матрицы кнопок с Blue Pill.  
Позволяет отлаживать прошивку без физического железа.

**Запуск:**
1. Открыть `button_box.pdsprj` в Proteus 8+
2. Загрузить `buttons.hex` в свойства микроконтроллера
3. Запустить симуляцию (F12)
